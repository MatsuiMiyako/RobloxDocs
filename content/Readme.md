# Zero to Game: A Beginner's Roblox Studio Guide

Welcome. This isn’t just another tutorial — this is the guide I wish I had when I first opened Roblox Studio and had no idea what anything meant.

## Why I Built This

I started learning Roblox development as a student, frustrated by how confusing and scattered most beginner tutorials were. I wanted:

- A **step-by-step** guide, not just random YouTube videos.
- Real explanations, not just "copy this code."
- A place where I could **actually learn to build**, not just follow instructions blindly.

So I built this guide from scratch — not to compete with the official documentation, but to make something that helps beginners actually understand and create.

## What This Project Covers

This guide teaches you:

- How to use Roblox Studio: UI, parts, models, properties, Explorer
- How to write your first real code: variables, conditionals, loops, functions
- How Roblox scripting works in plain English

All organized into levels that build your skill gradually.

## Who This Is For

- Total beginners
- Students or hobbyists trying to make their first game
- People who get stuck in tutorials and want real understanding

## What Makes This Different

- **Step-by-step flow** with a clear path from Level 0 upward
- **Assignments** to apply what you learn
- **Clean explanations** with screenshots and sample code
- A beginner’s lens — because I was one, and I still remember what confused me

## How to Use This

1. Start from [Level 0](https://matsuimiyako.github.io/RobloxDocs/Devlog/Level-0/)
2. Read each concept page.
3. Do the assignment at the bottom.
4. Build something small every time you finish a level.

### Final Note

This guide started as a way to teach myself. Now it’s my way of teaching others.

It’s open-source, free, and will keep evolving. If it helps even one other person avoid the pain I went through, it’s worth it.
