# Zero to Game

**A beginner's guide to Roblox Studio — from your first part to your first script.**

I built this as a student learning Roblox scripting from scratch — and I remember how overwhelming it was. This guide walks you through the fundamentals step-by-step, with code, visuals, and hands-on assignments.

- No experience required  
- Written for total beginners  
- Assignments to apply what you learn  

→→→ [Start Level 0 Now!](https://matsuimiyako.github.io/RobloxDocs/Devlog/Level-0/)

---

## Why I Made This

Roblox has official documentation — and it’s good — but it’s not built for beginners. Most tutorials either go too fast, assume too much, or just throw code at you without explaining anything.

This guide is different.
It’s built by a student, for students — with clarity, progression, and confidence in mind.

---

## Who This Is For

- Students learning game development  
- Anyone overwhelmed by Roblox Studio  
- Self-taught coders who want structure  

---

## What You'll Learn

- How to navigate the Roblox Studio interface  
- How Lua scripting works (with clean examples)  
- How to build real working game mechanics  
- How to think like a developer, not just copy code
