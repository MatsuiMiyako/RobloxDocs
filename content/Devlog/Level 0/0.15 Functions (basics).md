## Definition

A **function** is a reusable block of code that performs a specific task. You can call it multiple times from anywhere in your script without rewriting the same code.

```lua
function greet()
    print("Hello!")
end

greet()  -- calls the function
```

## Core Purpose

Functions allow you to:
- Avoid repeating yourself
- Break code into logical chunks
- Make scripts easier to read and debug
- Reuse logic with different inputs (parameters)
- Organize your game systems cleanly
## Syntax Structure

```lua
-- Define the function
function sayHello()
    print("Hello!")
end

-- Call (run) the function
sayHello()

```
With **parameters** (data you pass in):
```lua
function greetPlayer(name)
    print("Hello, " .. name)
end

greetPlayer("RobloxUser123")  -- Output: Hello, RobloxUser123
```
- Always use `function name()` to define  
- Always call it by name: `name()`  
- You can pass values (called arguments) into the function
## When to Use It

- Repeating the same logic (like printing, damaging, updating UI)
- Responding to events (touch, click, key press)
- Separating game systems into modules
- Improving code structure and reusability
## Common Mistakes / Gotchas

- Defining a function but never calling it → nothing happens
- Calling a function before it’s defined (Lua reads top-down)
- Forgetting to pass arguments when needed
- Using `return` incorrectly when you don’t need a result
## Code Example

```lua
-- This function starts a new wave in the game.
-- It doesn't take any input parameters.
function waveStart()
    -- Display a message in the console to indicate a new wave has started.
    print("A new wave has begun!")
end

-- This function gives coins to a specific player.
-- It takes two parameters:
-- 'playerName' is the name of the player receiving the coins.
-- 'amount' is the number of coins to give.
function giveCoins(playerName, amount)
    -- Combine strings and variables to print a message showing the coin reward.
    print(playerName .. " received " .. amount .. " coins!")
end

-- Call the waveStart function to begin a new wave.
waveStart()

-- Call the giveCoins function to give 50 coins to the player named "Pengda".
giveCoins("Pengda", 50)
```
