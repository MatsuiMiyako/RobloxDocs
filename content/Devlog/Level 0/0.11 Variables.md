# Concept: Variables

## Definition

A **variable** is a named container that stores a piece of data—like a number, word, or object. In Lua, you create a variable using the `local` keyword:
```lua
local coins = 10
```
Now `coins` holds the number 10, and you can reuse or change it later. The word `local` is used to declare that the **variable** is only used within that script, and that if you use the **variable** `coins` again, it wouldn't be holding the number 10, it would be different depending on the value you assign it. If you don't include the word `local` you would be declaring a `global` variable, which isn't good practice.
## Core Purpose

Variables are used to
- Store player stats (health, XP, score)
- Track states (isJumping, isDead, hasKey)
- Hold references to parts, UI, and tools
- Pass data between functions and scripts
They are the foundation of logic and memory in every system you build.

## Variable Types

| Type    | Example Syntax                 | Description                                |
| ------- | ------------------------------ | ------------------------------------------ |
| Number  | `local coins = 25`             | Whole or decimal numbers                   |
| String  | `local name = "RobloxUser123"` | Text (must be in quotes)                   |
| Boolean | `local isAlive = true`         | `true` or `false`                          |
| Object  | `local part = workspace.Part`  | Roblox instance (like a `part` or `model`) |

## When to Use It

- To save values between events
- To label objects so you don’t keep typing `workspace.ThisIsAGrayPartInTheMiddle`
- To customize behavior based on values
- To avoid “magic numbers” and hardcoded text

## Common Mistakes / Gotchas

- Forgetting `local` makes the variable global (which can break other scripts)
- Using the wrong data type (e.g., adding a number to a string)
- Overwriting a variable by accident
- Trying to use a variable before it’s declared

## Code Example

```lua
local playerName = "RobloxUser123"
local score = 50
local hasSword = true
local platform = workspace.Platform

print("Player: " .. playerName)
if hasSword then
    print("You are armed.")
end
```
