## Definition

A loop runs a block of code multiple times. In Lua, there are two main types:

- `for` loop → repeats a set number of times  
- `while` loop → repeats as long as a condition is true

Loops let you automate repetitive tasks and process lists without writing the same code over and over.


## Core Purpose

Loops are used to:
- Repeat a task (e.g., spawn 5 enemies)
- Go through items in a list (like players or objects)
- Run timed or conditional logic
- Build systems like countdowns, waves, animations, or cleanup routines

## Syntax Structure

```lua
-- FOR LOOP: Runs a block of code a specific number of times

-- This loop will repeat 5 times, starting at 1 and ending at 5
-- 'i' is the loop counter. It increases by 1 after each loop until it reaches 5
for i = 1, 5 do
    -- This line prints the current loop number
    -- '..' is used to join (concatenate) text with variables in Lua
    print("This is loop #" .. i)
end

----------------------------------------------------------------------------------------

-- WHILE LOOP: Repeats as long as a certain condition is true

-- We start with a variable called 'counter' set to 3
local counter = 3

-- This loop will keep running **as long as** 'counter' is greater than 0
while counter > 0 do
    -- Print the current value of 'counter'
    print("Counting down: " .. counter)

    -- Next, you decrease the value of 'counter' by 1 after each loop
    -- This is important so the loop doesn't run forever
    counter = counter - 1
end

-- When 'counter' becomes 0, the while loop ends

```
## When to Use It

- Looping through players: `for _, player in pairs(Players:GetPlayers())`
- Spawning multiple items at once
- Running a timer countdown
- Checking a condition every second (`while isRunning == true`)
- Repeating a process with minor changes each time
## Common Mistakes / Gotchas

- Infinite loops (a `while true do` with no exit = your game freezes)
- Forgetting to update the condition inside `while` loops
- Off-by-one errors (e.g., `1 to 5` loops 5 times, not 4)
- Trying to use `for` on things that aren’t numbers or lists

## Code Example

```lua
-- FOR loop to print round numbers from 1 to 3
-- Useful for iterating over a fixed number of times (e.g., game rounds)
for i = 1, 3 do
    print("Round " .. i)  -- Concatenate "Round" with the current round number and print
end

-- Initialize a variable to simulate player health
local health = 3

-- WHILE loop to simulate a health countdown
-- Continues as long as health is greater than zero
while health > 0 do
    print("Health: " .. health)  -- Print current health status
    health = health - 1          -- Decrease health by 1 (simulates taking damage)
end

-- Once the loop exits (health == 0), print a game-over message
print("You died.")  -- Final output when health reaches zero

```
