
# Concept: {{title}}

## Definition

The **Viewport** in Roblox Studio is the 3D scene where you place, select, and manipulate objects. It’s your “camera view” into the game world—everything you see during gameplay is visible and adjustable here.

## Core Purpose

Efficient **Viewport** navigation lets you:
- Move around the game world to position objects accurately
- Inspect builds, terrain, and character models
- Design maps, test scale, and place parts with precision

It’s how you actually build and see what you're scripting.

## Key Controls

| Action              | Control (Mouse & Keyboard)                      |
| ------------------- | ----------------------------------------------- |
| **Rotate camera**   | Right-click + move mouse                        |
| **Pan camera**      | Shift + right-click + drag OR middle mouse drag |
| **Zoom in/out**     | Scroll wheel                                    |
| **Focus on object** | Select object → Press `F`                       |
| **Move object**     | Use the Move/Scale/Rotate tools                 |
| **Select object**   | Click in Viewport or Explorer                   |
| **Reset camera**    | View tab → Reset View                           |
| **Select Part**     | Alt + left-click                                |
| **Move Camera**     | WASD keys                                       |
| **Select Tool**     | 1                                               |
| **Move Tool**       | 2                                               |
| **Scale Tool**      | 3                                               |
| **Rotate Tool**     | 4                                               |

## When to Use It

- Placing parts or models in specific world positions
- Adjusting terrain or building structures
- Inspecting if things are aligned, floating, or clipping
- Testing collision areas or hitboxes visually
- Seeing what the camera sees during gameplay

## Common Mistakes / Gotchas

- Using the **wrong tool mode** (e.g. Scale instead of Move)
- Losing your object—use `F` to find it again!
- Not locking the camera when trying to move parts
- Accidentally dragging terrain or environment pieces by misclicking
