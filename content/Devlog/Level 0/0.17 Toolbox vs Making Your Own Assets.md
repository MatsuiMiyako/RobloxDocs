# Concept: Toolbox vs Making Your Own Assets

## Definition

`Toolbox` is a built-in `Roblox` asset library with free models, scripts, and art. Making your own assets means building them yourself using parts, scripts, and UI elements.

## Core Purpose

This concept helps developers choose between:
- Speed (`Toolbox`)  
- Safety + control (DIY assets)

## Key Properties / Methods

- `Toolbox` tab in Roblox Studio
- Asset details panel (check for scripts or viruses)
- Manual modeling tools: `Part`, `Union`, `Mesh`, `Texture`, etc.

## When to Use It

- Use `Toolbox` for prototypes, ideas, or placeholder objects
- Use DIY assets for final builds, cleaner code, better performance

## Common Mistakes / Gotchas

- Importing free models with hidden scripts or viruses
- Not understanding the code that comes with `Toolbox` models
- Becoming dependent on free assets instead of learning the tools
