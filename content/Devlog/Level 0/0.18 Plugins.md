# Concept: Plugins

## Definition

**Plugins** are tools created by developers that add extra features to Roblox Studio.  
They can help you build maps, create animations, manage assets, and much more

## Core Purpose

- Speed up complex or repetitive tasks
- Give you better building, UI, or terrain tools
- Extend what Studio can do without scripting

## Common Plugins

- **F3X**: Advanced building tools
- **Moon Animator**: Animations
- **Archimedes Two**: Curved parts and shapes
- **Model Resize**: Scales models properly

## How to Use

1. Open the **Toolbox**, switch to the Plugins category
2. Click “Install” on a trusted plugin
3. Access it from the **Plugins** tab after install

## Common Mistakes / Gotchas

- Installing untrusted plugins that contain malicious code
- Relying on plugins without learning the underlying systems
- Forgetting to uninstall plugins you don’t use
