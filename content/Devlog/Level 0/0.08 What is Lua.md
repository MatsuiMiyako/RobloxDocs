
# Concept: What is Lua

## Definition

**Lua** is the programming language used to write game logic in Roblox. It’s lightweight, fast, beginner-friendly, and flexible—used to create everything from simple scripts to full multiplayer game systems.

Roblox uses a version called **Luau** (a Roblox-optimized fork of Lua), but the core syntax is very similar.


## Core Purpose

Lua powers everything interactive in a Roblox game:
- When a button is clicked
- When an enemy takes damage
- When a round starts
- When a player levels up
- When data is saved or loaded

Without Lua, your game is just static models and UI. With Lua, it becomes alive.

## Key Properties / Methods

- **Simple syntax** – Easy to read and write (no semicolons or curly braces)
- **Dynamically typed** – You don’t need to declare data types
- **Tables** – Core data structure (arrays, dictionaries)
- **Functions** – First-class, can be passed around like variables
- **Lightweight and fast** – Great for real-time game logic
- **Event-based** – You can respond to actions like touches or clicks

## When to Use It

You write Lua code in:
- **Scripts** (run on the server)
- **LocalScripts** (run on the client)
- **ModuleScripts** (shared logic you reuse across scripts)

Use it to:
- Build game mechanics
- Connect events (like when a player clicks a button)
- Control animations, UI, scores, enemies, inventory, etc.
- Communicate between server and client using RemoteEvents

## Common Mistakes / Gotchas

- Mixing up Lua with JavaScript, Python, or C-style languages
- Forgetting that Roblox uses a version called **Luau**, which adds new features
- Assuming you can run `print()` inside any object—only valid in Scripts or LocalScripts
- Trying to “drag-and-drop” everything without learning actual scripting

## Code Example

```lua
-- A basic Lua function that prints a message
function greetPlayer(name)
    print("Welcome, " .. name .. "!")
end

greetPlayer("RobloxUser123")  -- Output: Welcome, RobloxUser123!

-- A variable and if-statement
local score = 10

if score >= 10 then
    print("You passed!")
else
    print("Try again.")
end
```
